图标文件说明
===========

这个目录应当包含以下图标文件:
- check.png - 表示已完成翻译的条目
- empty.png - 表示未翻译的条目

图标文件要求:
1. 格式: PNG格式（推荐使用透明背景）
2. 尺寸: 16x16 或 24x24 像素
3. 颜色: 
   - check.png: 建议使用绿色(#4CAF50)或蓝色(#2196F3)
   - empty.png: 建议使用灰色(#9E9E9E)或红色(#F44336)
4. 文件大小: 建议小于10KB

获取图标的方法:
1. 从网上下载简单的对勾和叉号图标
   - 推荐网站: Material Icons, FontAwesome, Iconfont
   - 下载后请确保转换为正确的尺寸和格式

2. 使用在线图标库
   - Material Icons: https://material.io/resources/icons/
   - FontAwesome: https://fontawesome.com/icons
   - Iconfont: https://www.iconfont.cn/

3. 使用图像编辑软件创建
   - 推荐工具: Photoshop, GIMP, Figma
   - 创建时请使用透明背景
   - 导出时选择PNG格式

临时解决方案:
如果不提供这些图标文件，程序会在运行时显示警告，但不会影响基本功能。
程序会使用默认的文本标记（"✓" 和 "○"）来代替图标。
